
# 📚 Library Management System

A simple Java-based console application developed using Object-Oriented Programming concepts to manage library operations efficiently.

## 🚀 Features

- Add new books
- View available books
- Issue books
- Console-based menu system
- Modular package structure
- User input handling using Scanner

## 🛠️ Technologies Used

- Java
- IntelliJ IDEA
- OOP Concepts


## 📂 Project Structure

LibraryManagementSystem/
│
├── model/
│ └── Book.java
│
├── service/
│ └── LibraryService.java
│
├── util/
│ └── InputUtil.java
│
└── Main.java
